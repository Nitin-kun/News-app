# News App

A news application built with Flutter and Firebase, inspired by Inshorts.

## Getting Started

This project is a starting point for a Flutter application. It uses Firebase as the backend to manage user authentication and data.

## Features

- **User Authentication**: Log in with email and password or use Google for authentication. New users can sign up easily.
- **Home Screen**: After logging in, users are directed to the home screen displaying news articles.
- **Vertical Swipe**: Swipe vertically to view different articles.
- **Right Swipe**: Swipe right to open the categories section to choose new categories.
- **Left Swipe**: Swipe left to open the news source in an in-app browser.

## How to Use the App

1. **Login**: Log in with your email and password or use Google for quick access.
2. **Sign Up**: New users can sign up using the sign-up option.
3. **Home Screen**: After logging in, you will be directed to the home screen.
4. **Navigate Articles**: Swipe vertically to browse through different articles.
5. **Categories**: Swipe right to access the categories section and choose your preferred categories.
6. **In-App Browser**: Swipe left to open the news source in the in-app browser for a detailed view.
